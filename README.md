# 🐝 SKbossbea — v1.0
**Підтримувана версія гри:** `1.16.5+`  
**Залежності:** `WorldEdit`, `WorldGuard`

---

### 📖 Про плагін
**SKbossbea** — це преміальний скрипт для створення динамічної події "Бос-Бджола". Це не просто спавн моба, а повноцінний івент із автоматичною зміною ландшафту, захистом території та унікальними бойовими механіками.

Коли настає час, плагін автоматично будує арену за допомогою схематики, створює безпечну зону (Region) та закликає розлючених бджіл-босів із величезним запасом здоров'я.

---

### ✨ Ключові особливості

* **🏗 Автоматична Арена (Schematic)**
  Плагін використовує WorldEdit API для вставки схематики (`.schem`) під час старту івенту та автоматично прибирає її після завершення.
* **🛡 Інтеграція з WorldGuard**
  Під час івенту автоматично створюється захищений регіон `bossbea` із високим пріоритетом, що забороняє гравцям ламати блоки арени або будувати на ній.
* **⚔️ Унікальна механіка атаки**
  Бджоли-боси мають здатність **підкидати гравця в повітря** при кожному ударі, що робить бій набагато складнішим та динамічнішим.
* **⏰ Розклад та Таймери**
  Можливість налаштувати точний час запуску (наприклад, `14:00`, `18:00`) прямо в конфігу. Також є система автоматичного завершення івенту через 30 хвилин, якщо боса не вбили.
* **📍 Розумне повернення (Anti-Leash)**
  Якщо бджола відлітає занадто далеко від центру івенту (більше ніж на 20 блоків), вона миттєво телепортується назад, що запобігає "виведенню" боса з арени.
* **🎁 Нагороди через консоль**
  Гнучке налаштування винагороди за вбивство боса через будь-яку консольну команду.

---

### 💻 Командний центр

| Команда | Опис | Доступ (Permission) |
| :--- | :--- | :--- |
| `/bossbea delay` | Дізнатися час до наступного івенту | `bossbea.player` |
| `/bossbea setspawn` | Встановити точку спавну івенту | `bossbea.admin` |
| `/bossbea start` | Примусово запустити боса зараз | `bossbea.admin` |
| `/bossbea reload` | Перезавантажити налаштування | `bossbea.admin` |

---

### 🛠 Встановлення
1. Переконайтеся, що на сервері встановлені **WorldEdit** та **WorldGuard**.
2. Помістіть файл плагіна в папку `plugins`.
3. Створіть у папці `plugins/SKbossbea/schematics/` файл з назвою **`bossbea.schem`** (або плагін створить стандартний при першому запуску).
4. Налаштуйте час запуску в `config.yml`.
5. У грі станьте на місце майбутньої арени та пропишіть `/bossbea setspawn`.
